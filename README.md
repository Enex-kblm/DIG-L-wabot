# DiG-L-wabot

## Run Locally

Clone the project

```bash
  git clone https://github.com/Enex-kblm/DIG-L-wabot.git
  cd DIG-L-wabot
  npm install
  npm start
```

Install dependencies **recommended**
```bash
 npm install --force
```
